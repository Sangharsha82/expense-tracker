# Expense tracker
A simple web tracker for your expenses

## Installation
1. Clone the repository
```bash 
git clone https://github.com/Sangharsha82/expense-tracker
```
2. Open  `index.html` in your browser

## Usage 
1. Enter **amount** , pick a **category** and add optional **category**
2. Click **submit** to save your changes 
3. Click **delete** to remove expense.
